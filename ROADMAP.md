# Roadmap

## Phase 1: Foundation

Goal: create the project infrastructure needed for long-term, source-driven research.

Milestones:

- [x] Initialize public research repository.
- [x] Initialize private archive repository.
- [x] Create initial bibliography and acquisition tracker.
- [x] Create project dashboard.
- [x] Create agent/contributor guidance.
- [x] Create decision log.
- [x] Create research log.
- [x] Create governance files.
- [x] Add issue templates.

## Phase 2: Port Breton Source Acquisition

Goal: acquire and catalog all legally mirrorable core sources for the Port Breton case.

Milestones:

- [ ] Locate and mirror *La Nouvelle France* issues where legally permitted.
- [ ] Locate and mirror Paul de Groote's promotional book where legally permitted.
- [ ] Locate and mirror Baudouin's 1883 account where legally permitted.
- [ ] Identify and catalog French legal/trial records.
- [ ] Search and catalog Trove newspaper coverage.
- [ ] Locate relevant nineteenth-century maps.
- [ ] Create source notes for each acquired source.

## Phase 3: Evidence Model

Goal: convert source material into structured research records.

Milestones:

- [ ] Define schemas for people, ships, places, events, organizations, and claims.
- [ ] Build Port Breton people index.
- [ ] Build Port Breton ships index.
- [ ] Build Port Breton places index.
- [ ] Build initial cited timeline.
- [ ] Build claim register for disputed or important facts.

## Phase 4: Analysis

Goal: synthesize evidence into historically useful interpretations without losing source traceability.

Research themes:

- recruitment and propaganda
- finance and speculation
- religion and institutional trust
- migration networks
- voyage logistics
- disease, mortality, and abandonment
- trial and public scandal
- survivor settlement and New Italy
- memory and historiography

## Phase 5: Publication

Goal: make the research useful to readers and future researchers.

Possible outputs:

- annotated bibliography
- public research website
- narrative essays
- interactive timeline
- map of voyages and settlements
- English-language source guide
- long-form narrative history
