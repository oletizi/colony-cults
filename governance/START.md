# Session Start

Start every session by restoring project context from repository files.

## Read first

1. `PROJECT.md`
2. `ROADMAP.md`
3. `DECISIONS.md`
4. `RESEARCH_LOG.md`
5. `GOVERNANCE.md`
6. `governance/STATE.yaml`
7. `governance/NEXT.md`

## Then do this

1. Confirm the current phase and milestone.
2. Identify the highest-priority unfinished task.
3. Check whether any prior decision constrains the task.
4. Prefer a small recoverable change over a large uncommitted batch.
5. If the session changes project priorities, update the durable files.

## Startup rule

Do not rely on memory when the repository already contains the answer.
