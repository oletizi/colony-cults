# Checklists

## Session start checklist

- Read the required dashboard and log files.
- Confirm current milestone and highest-priority unfinished task.
- Check for prior decisions that constrain the work.
- Choose a small recoverable unit of progress.

## Session end checklist

- Update the research log.
- Record durable decisions.
- Refresh state and next actions.
- Commit.
- Push.

## Research integrity checklist

- Separate evidence from interpretation.
- Attach or prepare source IDs for factual claims.
- Preserve conflicting claims when sources disagree.
- Mark uncertainty explicitly.
- Do not mirror restricted material.

## Recovery checklist

- If context is lost, restart from repository files.
- If work is partial, commit the partial state with a clear message.
- If priorities shift, update the durable files before stopping.
