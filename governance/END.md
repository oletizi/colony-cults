# Session End

End every session by leaving durable state behind.

## Before stopping

1. Update `RESEARCH_LOG.md`.
2. Record durable decisions in `DECISIONS.md` if any were made.
3. Refresh `governance/STATE.yaml` if the active focus changed.
4. Refresh `governance/NEXT.md` so the next session has a clean handoff.
5. Commit the work in a small coherent unit.
6. Push the commit.

## Exit rule

Do not leave important project state trapped in memory, scratch notes, or chat history.
